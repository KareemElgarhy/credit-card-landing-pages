# Credit Card Landing Pages

This project contains two responsive landing pages designed for credit card offers.  
The goal of the project is to simulate a real-world freelance task focused on **lead generation and conversion-focused design**.

## Project Overview

The project includes two landing pages:

### Landing Page 1 – Lead Capture Page

A marketing landing page designed to collect user information through a simple form.

Features:
- Financial themed design
- User input form
- Clean two-column layout
- Trust signals and call-to-action

Form Fields:
- Gender
- First Name
- Last Name
- Email

Purpose:
Capture leads for credit card applications.

---

### Landing Page 2 – Survey Style Funnel

A survey-style landing page inspired by a conversion funnel layout.

Features:
- Question / answer interaction
- Step-based survey flow
- Call-to-action navigation
- Conversion focused layout

Purpose:
Guide users through a simple qualification flow before presenting a credit card offer.

---
